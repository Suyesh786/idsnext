Queue FIFO – First In First Out

What is a Queue?

A queue is a linear data structure that follows the principle of FIFO (First In First Out).

The first element inserted into the queue is the first one to be removed — just like a real-world queue of people waiting in line.

Basic Operations

  • Enqueue – Insert an element at the rear of the queue
  • Dequeue – Remove the element from the front of the queue
  • Peek / Front – View the front element without removing it
  • IsEmpty – Check if the queue is empty
  • Size – Return the number of elements in the queue

Example

Enqueue 1
Enqueue 2
Enqueue 3

Queue (front → rear):
1  2  3

Dequeue → removes 1

Queue (front → rear):
2  3

Real-World Analogies

  • A line of customers at a ticket counter
  • Print jobs sent to a printer
  • CPU task scheduling
  • Breadth-First Search (BFS) in graphs

Time Complexity

  • Enqueue – O(1)
  • Dequeue – O(1)
  • Peek – O(1)
  • Search – O(n)
